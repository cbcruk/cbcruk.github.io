> `field-sizing`를 사용하면 콘텐츠를 기반으로 크기 조절을 사용 설정하는 데 CSS 한 줄이 필요합니다. 이 콘텐츠 기반 크기 조절 스타일은 `textarea` 외에도 다른 요소에도 적용됩니다.

<textarea style={{ padding: '10px', fieldSizing: 'content' }}>{`textarea, select, input {
\tfield-sizing: content;
}`}</textarea>
