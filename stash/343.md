`delete` 연산자는 객체에서 속성을 제거하는 데 사용되며, 변수나 기본값은 삭제하지 않음.

## 주요 사항

```js
// 1. 자신의 속성만 삭제
const obj1 = { name: 'Alice', age: 25 }
console.log(delete obj1.age) // true

// 2. 존재하지 않는 속성 삭제
const obj2 = { name: 'Alice' }
console.log(delete obj2.city) // true

// 3. `configurable` 속성
const obj3 = Object.defineProperty({}, 'readOnly', {
  value: 'I cannot be deleted',
  configurable: false,
})
console.log(delete obj3.readOnly) // false

// 4. 전역 변수 및 함수
let globalVar = 'I exist'
console.log(delete globalVar) // false

// 5. 배열 요소
let arr = [1, 2, 3]
console.log(delete arr[1]) // true
console.log(arr) // [1, <empty>, 3]

// 6. 메모리 관리
let obj4 = { key: {} }
delete obj4.key // 참조 사라짐
```

## 결론

- `delete`는 **설정 가능한** 속성에만 사용하고, 설정 불가능한 속성이나 전역 변수를 삭제할 때 주의할 것.
- 객체 참조 삭제 후, 가비지 컬렉터가 메모리 해제.
